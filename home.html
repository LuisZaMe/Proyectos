Pasos para imagen Hello world!

  1. Sirve para buscar la versión oficial de Ubuntu
    2. docker search ubuntu
  2. Descargar la versión que necesitemos
    3. docker pull ubuntu
  3. Ejecutar Ubuntu
    4. docker run -it ubuntu
  4. Lista de contenedores / -a para ver el historial
    5. docker ps / docker ps -a

Pasos para descargar imagen NGINX y crear un sitio web

  1. Descargar NGINX
    2. docker pull nginx
  2. Ejecutar NGINX
    3. docker run -it nginx
  3. Ejecutar NGINX en un puerto en específico -d (detach) sirve para apagar el log del contenedor
    4. docker run -p 4000:80 -d nginx
  4. Lista de contenedores / -a para ver el historial
    5. docker ps / docker ps -a
  5. Sirve para parar todos los contenedores. ¡ALERTA! Este comando detendrá todos los contenedores.
    6. docker stop $(docker ps -aq)
  6. Formato de salida (opcional)
    7. docker ps --format="ID\t{{.ID}}\nNAME\t{{.Names}}\nPORT\t{{.Ports}}\nStatus\t{{.Status}}\nCOMMAND\t{{.Command}}\nCREATED\t{{.CreatedAt}}\nSize\t{{.Size}}\n"
  7. Para copiar los archivos (Run as Administrator)
    8. docker run -d -p 4000:80 --name website -v ${PWD}:/usr/share/nginx/html:ro -d nginx : tiene Read-only activado docker run -d -p 4000:80 --name website -v ${PWD}:/usr/share/nginx/html -d nginx

Pasos para crear una imagen personalizada.

  1. Generar una imagen desde un archivo DockerFile.
    2. docker build -t luis-nginx .
  2. Una vez que la imagen es creada ejecutar para desplegar el sitio web.
    3. docker run -d -p 4000:80 --name chamacomalowebsite luis-nginx
